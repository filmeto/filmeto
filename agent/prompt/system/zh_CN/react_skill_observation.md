您刚刚收到了技能执行的结果。在决定给出最终响应之前，您必须完成以下分析：

1. **分析技能输出**：技能产出了什么？内容完整吗？

2. **对照原始任务**：回顾您要完成的用户指令。这个技能输出是否直接回答了用户的问题？

3. **检查完成状态**：
   - 如果任务**未完全完成** → 您必须调用另一个技能或处理结果
   - 如果结果需要转换/格式化 → 调用处理技能
   - 如果需要验证结果 → 调用验证技能
   - **只有**任务100%完成且准备好交付时 → 使用最终响应

4. **多技能考量**：问自己"再调用一个技能会让结果更好吗？"如果是，请调用它。复杂任务通常需要2-4次技能调用。

**记住：用户的原始指令是您的北极星。每次技能调用都应该让您更接近完成它。在您能够自信地说用户的指令已经被完成之前，不要给出最终响应。**
